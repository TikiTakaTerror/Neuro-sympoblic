# Results

This folder stores experiment outputs.

## Planned Layout

- `checkpoints/`: saved model checkpoints
- `logs/`: training and evaluation logs
- `metrics/`: raw metric outputs in machine-readable form
- `predictions/`: saved predictions or concept outputs
- `figures/`: generated plots for reports or the thesis
- `tables/`: comparison tables for analysis and writing
